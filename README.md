# Grama-Vaxi 🐄💉

Grama-Vaxi is a bilingual Android-based village animal healthcare and vaccination reminder application developed to support farmers and rural communities in managing livestock health efficiently. The application helps users register animals, track vaccination schedules, receive healthcare reminders, and report sick animals quickly through a simple and user-friendly interface.

The application supports both **English** and **Kannada** languages to improve accessibility and usability for rural users.

---

## 📌 Features

* 🌐 Bilingual Support (English & Kannada)
* 🐄 Animal Registration System
* 💉 Vaccination Reminder Tracking
* 🚨 Sick Animal Reporting
* 🔐 User Registration & Login
* 📋 Animal Records Management
* 📱 Simple and Rural-Friendly UI

## 📂 Project Modules

### 1. Language Selection Module

Allows users to choose English or Kannada language during app startup.

### 2. User Authentication Module

Provides basic user registration and login functionality.

### 3. Animal Registration Module

Stores animal details such as:

* Animal Name
* Type/Breed
* Owner Information
* Vaccination Details

### 4. Vaccination Reminder Module

Helps farmers remember upcoming vaccination schedules and healthcare alerts.

### 5. Sick Animal Reporting Module

Allows users to report unhealthy animals and notify veterinary support.
